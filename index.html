Кратко об проекте

Музыкальный каталог - информационная система, позволяющая пользователям просматривать большое количество информации о музыкальных композициях, добавлять их плейлист для просмотра через некоторое время, а также предоставляющая возможность редактирования всех основных данных из приложения.

Как разрабатывалась

Информационная система реализована в формате сайта. Она делится на две части: бэкенд (сервер) и фронтенд (веб-приложения). Бэкенд отвечает за обработку данных и сложных вычислений, фронтенд является графическим клиентом для данных, поступающих с бэкенда. Бэкенд реализован в виде Web API приложения, написанного на языке программирования C#, а точнее на фреймворке ASP.NET. В качестве среды разработки была выбрана IDE Microsoft Visual Studio. ASP.NET — это часть технологии .NET, используемая для написания мощных клиент-серверных приложений.

Фронтенд реализован в виде одностраничного веб-приложения (SPA). Он написан на языке программирования JavaScript, а точнее на фреймворке Vue.js. Разработка фронтенда была произведена в среде разработке PhpStorm. Vue — это прогрессивный фреймворк для создания пользовательских интерфейсов. В отличие от фреймворков-монолитов, Vue создан пригодным для постепенного внедрения. Vue полностью подходит для создания сложных одностраничных приложений.

В роли системы управления базой данных в данном проекте выступает Microsoft SQL Server. Это одна из популярнейших реляционных СУБД, входящая в пакет программ, разработанных Microsoft. В качестве интегрированной среды для управления базой данных, было выбрано приложение Microsoft SQL Server Management Studio 18.

Интерфейс веб-приложения был спроектирован в программе Figma.
